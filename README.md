# Anghami Plus

Chrome extension for `kalimat.anghami.com` that adds:

- Page cleanup (hides clutter, keeps lyrics + album art)
- Better Arabic font (Amiri / Noto Naskh Arabic)
- English translation (via AWS Bedrock)
- Harakat toggle

See `INSTRUCTIONS.md` for implementation spec and `ARCHITECTURE.md` for AWS design.
